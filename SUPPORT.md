# Support

Need help using **Open Source Project Polish**? Here's where to go.

## Questions & discussion

- Open a **GitHub Issue** with the `question` label for usage questions.
- Check the [README](README.md) and [SKILL.md](SKILL.md) first — most usage
  details are documented there.

## Reporting bugs

Open an issue using the **Bug report** template. Helpful details:

- The prompt you gave the agent.
- What the skill did vs. what you expected.
- The project type/state it ran against (git or not, existing docs, etc.).

## Requesting features

Open an issue using the **Feature request** template and describe the workflow
you'd like the skill to handle.

## Security issues

Do **not** file security problems as public issues. See [SECURITY.md](SECURITY.md).
